HOW TO START WEB UI
  1. Go to the HW client with Mongoose deployed
  2. Start Mongoose in webui mode $ java -jar <path to jar>/mongoose.jar webui
  3. Open a browser on your PC and open Mongoose Web UI page. Web UI address is <HW client IP>:8080. For instance, for Mongoose started locally the address would be http://localhost:8080/

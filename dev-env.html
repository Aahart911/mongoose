DEVELOPMENT ENVIRONMENT

SOFTWARE
  • JDK 7+
  • Git
  • Gradle
  • Text Editor or IDE

GET THE SOURCES

Go to the directory where you want to have Mongoose Git repository and execute the following command:

git clone https://github.com/akurilov/mongoose.git

This will create "mongoose" directory containing the project source code and other supplementary files

GRADLE

Build

Go to the "mongoose" directory and try the following command:

./gradlew clean dist

After gradle reports "build successful", you can find the tgz-file generated in build/dist directory.

Prepare IntelliJ IDEA project

./gradlew clean idea

Run the unit tests

./gradlew clean unitTest

Run the system tests

./gradlew clean systemTest

Note System tests execution is a heavy task so it may use a few gigabytes of memory and up to 1 hour
